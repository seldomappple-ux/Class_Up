## 组件规范

### 面板 `.panel`

- 背景白，边框 `#f0d9b0`，圆角 `10px`，阴影 `0 2px 6px rgba(180,120,40,.07)`
- 标题 `.panel-title`：橙色 `#c2600a`，12px，全大写，字间距 `.06em`

- 虚线边框 `#e8c98a`，hover/drag 时边框变 `#e07820`，背景 `#fff5e6`
- 文件选中后显示文件名（橙色加粗）

### 输入框 / 下拉

- 边框 `#e8c98a`，背景 `#fffdf7`
- focus：边框 `#e07820`，外发光 `rgba(224,120,32,.12)`

### 按钮

- 主按钮：橙色渐变，白色文字，圆角 `7px`
- 下载按钮：透明背景，边框 `#e8c98a`，文字 `#6b4226`

### API Key 输入

- 密码框 + 右侧 👁 切换明文按钮
- 切换后图标变 🙈

## 功能模块

### 1. 视频转音频面板

- 上传视频 → 选择采样率（默认 16000）和声道（默认单声道）→ 提交
- 成功后显示下载 WAV 按钮
- API：`POST /audio`，返回 `{ audio_id, filename }`；下载 `GET /audio/{id}/download`

### 2. 新建 M1 转录任务面板

- 上传视频 + 配置参数 → 提交
- provider 选项：`mock`（无需 Key）/ `豆包语音识别`（必填 Key）/ `自定义远程服务`（显示模型名和 Endpoint 字段）
- 选择豆包时自动填充：model=`Doubao-录音文件识别2.0`，endpoint=`https://openspeech.bytedance.com`，并发数上限 2
- API Key 通过 `localStorage`（key：`class_up_api_keys`，对象按 provider 分键）跨会话记忆
- API：`POST /jobs`（multipart），返回 `{ task_id }`

### 3. M1 转录进度面板

单条大进度条（高 28px，圆角 14px），进度条内显示当前状态文字。

**进度推导逻辑**（从 manifest 数据派生，无需后端额外字段）：

| 阶段 | 触发条件 | 进度值 | 标签 |
|---|---|---|---|
| 等待中 | m1 未 running | 0% | 等待中 |
| 提取音频 | m1=running，无 normalized_audio | 流光动画 | 提取音频... |
| 音频分段 | 有 audio，无 segments | 12% | 音频分段中... |
| 准备识别 | 有 segments，done=0 | 18% | 准备识别分段... |
| 语音识别 | done < total | 20%~75%（按段数比例） | 语音识别中... |
| 合并结果 | 全段完成，无 outputs | 80% | 合并识别结果... |
| 生成字幕 | 有部分 output | 92% | 生成字幕文稿... |
| 完成 | m1=success | 100% | 转录完成 ✓ |
| 失败 | m1=failed | 100%（红色） | 转录失败 |

- 运行中无法确定精确进度时使用流光动画（`indeterminate`）
- 语音识别阶段在进度条下方显示副文字：`N / 总段 完成，M 段进行中`
- 轮询间隔：2 秒，`GET /jobs/{task_id}`

### 4. 输出预览面板

- 任务成功后显示
- SRT 前 20 行 + 文稿前 500 字（等宽字体预览框）
- 下载按钮：SRT / 文稿
- 下载文件名使用 manifest 中的真实输出文件名，例如 `{原文件名}_Subtitles.srt` 和 `{原文件名}_text.txt`；重复转录时展示带数字后缀的实际文件名。

### 5. 清理管理面板

- 提供清理预览按钮：展示候选删除项、类型、原因、预计释放空间。
- 提供执行清理按钮：只在用户确认后调用真实删除接口。
- 清理结果展示成功/失败数量、释放空间和失败 warning。
- 默认不提供删除最终成果的开关。

### 6. 侧边栏历史任务

- 页面加载时拉取 `GET /jobs`，按 `created_at` 倒序
- 点击任务条目切换当前任务并开始轮询
- 显示任务标题（截断）+ 状态徽章

## API 接口汇总

| 方法 | 路径 | 用途 |
|---|---|---|
| `POST` | `/jobs` | 创建转录任务（multipart） |
| `GET` | `/jobs` | 历史任务列表 |
| `GET` | `/jobs/{id}` | 任务详情（含 manifest snapshot） |
| `GET` | `/jobs/{id}/file/full_subtitle` | SRT 文件内容 |
| `GET` | `/jobs/{id}/file/full_transcript` | 文稿文件内容 |
| `GET` | `/cleanup/preview` | 清理预览，返回候选删除项和预计释放空间 |
| `POST` | `/cleanup/run` | 执行清理，返回删除结果摘要 |
| `POST` | `/audio` | 视频转音频（multipart） |
| `GET` | `/audio/{id}/download` | 下载 WAV |
| `POST` | `/shutdown` | 关闭服务（`beforeunload` 触发） |
