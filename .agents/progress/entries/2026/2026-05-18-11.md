Implemented the M1 backend foundation for the course assistant project.

The implementation adds the Python package, CLI entrypoint, configuration sample, manifest handling, ffmpeg wrapper, audio segmentation contract, transcription service interface, mock transcription service, merge logic, SRT rendering, transcript output, and focused tests.

The first runnable backend path intentionally uses the `mock` transcription provider. Real domestic or specialized model integration remains behind the transcription service interface and must be added after the concrete request and response protocol is known.

Verification completed:

- `python -m pytest`
- `$env:PYTHONPATH='src'; python -m class_up.cli --help`
- `$env:PYTHONPATH='src'; python -c "from class_up.config import load_config; c=load_config('config/config.example.yaml'); print(c.transcription.provider, c.media.segment_seconds)"`

Environment note: this machine currently has no `ffmpeg` or `ffprobe` on PATH, so real video extraction was not manually executed in this round.
