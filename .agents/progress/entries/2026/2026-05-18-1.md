Imported the minimum lightweight governance skeleton from the Taotaotao lightweight-general template guidance into this project.

The imported scope is limited to `.agents/` source files, local overrides, progress templates, and a yearly progress entry location. Taotaotao's historical progress entries, generated `.managed` files, IDE extension folders, and generated root adapter files were intentionally not copied.

This entry records the template import itself so future handoff can tell when the governance layer was introduced and which source guidance was used.
