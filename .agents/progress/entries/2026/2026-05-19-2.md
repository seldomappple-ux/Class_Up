# 20260519-2: Add FastAPI Web UI for M1 Pipeline

## Date
2026-05-19

## Status
draft

## Summary
为 M1 后端流水线新增 FastAPI + 原生 HTML 的本地 Web UI，支持视频上传、实时进度轮询、输出预览和下载，以及关闭页面时自动停止服务。

## Changes

### New Files
- `src/class_up/api/__init__.py` — API 包初始化
- `src/class_up/api/app.py` — FastAPI 应用，路由定义，`start()` / `start_mock()` 启动函数，`POST /shutdown` 端点
- `src/class_up/api/jobs.py` — 任务管理：构建配置、运行 M1 流水线、查询/列出任务
- `static/index.html` — 单页面前端，深色主题，侧边栏历史任务，表单 + 进度 + 预览 + 下载
- `start.bat` — 一键启动（正式模式）
- `start_mock.bat` — 一键启动（mock 模式，无需 API Key）

### Modified Files
- `pyproject.toml` — 新增 `web` 可选依赖组（fastapi, uvicorn, python-multipart），新增 `class-up-web` 和 `class-up-mock` 入口点

## API Design
```
POST /jobs          上传视频，后台启动 M1 流水线
GET  /jobs          列出所有历史任务
GET  /jobs/{id}     查询任务状态（manifest snapshot）
GET  /jobs/{id}/file/{key}  获取输出文件内容（full_subtitle / full_transcript）
POST /shutdown      关闭服务进程
```

## Behavior Notes
- 前端通过 `beforeunload` + `navigator.sendBeacon('/shutdown')` 实现关闭页面时自动停止后台服务
- `start_mock()` 自动设置 `CLASS_UP_TRANSCRIPTION_API_KEY=mock`，无需填写真实 Key
- bat 文件使用 `.installed` 标志文件跳过重复安装，首次运行后启动速度正常
- 视频上传保存至 `outputs/uploads/{uuid}/video.*`，输出目录沿用 M1 标准结构
