Added `docs/数据协议.md` as the detailed data contract for the course assistant workflow.

The protocol covers configuration, output layout, manifest structure, audio segments, transcription results, merged subtitles, transcripts, knowledge points, Markdown outputs, video clips, filename safety, stage handoff contracts, schema versioning, and minimum validation checks.

This document is intended to support implementation across M1, M2, and M3 without binding the project to a fixed model provider.
